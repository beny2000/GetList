For the Tech Enthusiasts

If you're curious about the technical details behind the scenes, here's a glimpse:

Backend Technologies

The app's backend is powered by Python, FastAPI, and Docker. These technologies work together seamlessly to provide a robust and scalable foundation for the app's functionalities.

Frontend Technologies

The frontend is crafted with JavaScript, React Native, and Expo, ensuring a responsive and intuitive user interface.

API Integration

We leverage the Google Places API to enhance location-based features, providing accurate and up-to-date information.

Machine Learning Magic

The app employs BERT (Bidirectional Encoder Representations from Transformers), a cutting-edge machine learning model from Huggingface, to intelligently classify items by store types and suggest relevant nearby stores.

Database Delight

MongoDB, equipped with geospatial search functionality, handles the storage and retrieval of geolocation data for stores and items.

DevOps Dance

Docker takes center stage for containerization, ensuring a consistent and scalable deployment environment for our app.
